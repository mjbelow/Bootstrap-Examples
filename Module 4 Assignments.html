Part A:

Select one of the Bootstrap templates from the "Getting Started with Bootstrap" tutorial examples and create an opening page for a traveling symphony orchestra. This exercise is to familiarize you with Bootstrap and let you practice a little bit with your design skills on a "random" project. You only have to create one page and should use any relevant photos you would like (cite sources/give credit), but you must create the text and the layout.

For full credit, this part of the lab must:

  • Use Bootstrap extensively. Using CSS/Less/HTML is allowed, but the main focus of this lab is Bootstrap and any lab that does not use it will not receive credit (even if the page has all the necessary content and style).
  • Include several images. Please be sure to cite sources.
  • Include a tour schedule. This should be a neatly organized table. The table should be responsive (grading will include window resizing, and credit for this part of the lab will depend partly on how your table responds).

Part B:

This is a continuation of the car dealership survey assignment. Use Bootstrap to create a simple main page for the dealership. Use Bootstrap extensively. The page should include the following elements:

  • Must be consistent with overall theme (colors, appropriate alignment, etc.)
  • Include at least three images of vehicles available at the dealership.
  • Include information about each vehicle, including pricing, vehicle technical details, and any other information that might be useful to the site's user.
  • A "Contact Online Sales" section -- make up contact info, don't use anything real.
  • A "Take Online Survey" section -- this should link up with/utilize your Lab 3 work (or Lab 2b, if you prefer).

Note that there are quite a few different ways to implement this page. What Bootstrap elements you choose to use is up to you, however, please try to make your page look polished and professional (and also consistent with your other pages -- survey and contact online sales).
